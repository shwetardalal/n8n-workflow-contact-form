# Contact Form Automation using n8n

This project is an automated contact form workflow built using n8n.

## Workflow Overview

The workflow automates the complete process of handling form submissions:

1. User submits a contact form
2. Data is captured automatically
3. Form details are stored in Google Sheets
4. Confirmation or notification email is sent using Gmail

## Workflow Architecture

Form Submission → Google Sheets → Gmail Notification

## Technologies Used

* n8n
* Google Sheets API
* Gmail Integration
* Workflow Automation

## Features

* Automated form handling
* Real-time data storage
* Automatic email notifications
* No manual data entry required
* Easy to customize and scale

## Workflow File

* `Contact Form Automation.json`

## Use Cases

* Portfolio websites
* Business inquiry systems
* Client lead management
* Student project demonstrations
* Automation practice projects

## How to Use

1. Import the workflow JSON file into n8n
2. Configure Google Sheets credentials
3. Configure Gmail credentials
4. Activate the workflow
5. Test the contact form submission
